Module 0

Introduction to calcium imaging

Calcium imaging is used in neuroscience as a proxy for neural activity, since calcium acts as a secondary signaling molecule in neural function. The objective of this module is to introduce biological and technical fundamentals before data processing.

0.1 How does calcium imaging work?

  • Host tissue: neurons in vivo or cultured systems.
  • Calcium-sensitive sensors: genetically encoded or chemical indicators.
  • Optical system: microscope and fluorescence detection setup.
  • Signal processing: extracting and interpreting fluorescence dynamics.

0.2 Understanding calcium signals in neurons

We focus primarily on somatic calcium signals while emphasizing how dendritic signals capture synaptic and compartment-specific information.

0.3 Why calcium and neural dynamics?

Systems-level neuroscience depends on linking population activity to behavior and identifying recurring ensemble-level motifs across state transitions.

Module 1

Exploring somatic calcium dynamics

This module introduces practical signal processing steps using extracted traces.

Task 1.0 Import data

  • Expected dimensions: 324 neurons x 15000 frames.
  • Recorded at 4.8 frames per second.

Task 1.1 Signal cleaning

  • Normalize with DeltaF/F.
  • Remove slow background trends.
  • Denoise while preserving transient events.

Task 1.2 Detect transient events

  • Event thresholding and trace-level validation.
  • Event raster visualization across neurons and time.
  • Feature extraction (rates, distributions, custom metrics).

Task 1.3 Ensemble features

  • Coactivity in short bins (for example 500 ms).
  • Null distributions from shuffled event trains.
  • Jaccard similarity and clustering of ensemble events.
