Module 0

Introduction to calcium imaging

Calcium imaging is used in neuroscience as a proxy for neural activity because calcium acts as a key secondary signaling molecule in neural function. The method space is broad and can be hard to enter, so this tutorial is designed to smooth that learning curve.

0.1 How calcium imaging works

  • Host tissue: neurons in cultured systems or live animals.
  • Calcium indicators: chemical or genetically encoded sensors.
  • Optical imaging: 1-photon or 2-photon excitation systems.
  • Signal processing: extracting and interpreting fluorescence dynamics.

Recommended background review: Grienberger and Konnerth.

Signal Context

Somatic vs dendritic signals

Somatic calcium imaging provides robust readouts of action-potential related calcium influx, while dendritic imaging is critical for resolving synaptic-input structure and local processing.

Useful review: Ali and Kwan.

Systems motivation review: Yuste: neuron-to-network perspective.

Module 1

Exploring somatic calcium dynamics

This module walks through practical processing tasks starting from extracted calcium traces.

Task 1.0: Importing data

Use the provided fluorescence dataset: Fluorescent Data.csv.

  • Dimensions: 324 neurons x 15000 frames.
  • Signal: raw mean intensity from GCaMP6s neurons.
  • Sampling rate: 4.8 Hz.

Task 1.1: Signal cleaning

  • Normalize using DeltaF/F = (F[t] - F0) / F0.
  • Subtract slow background fluctuations (>20 s scale).
  • Denoise traces (wavelet denoising is a common strategy).

Deliverable: show representative traces before/after each step.

Task 1.2: Detect transient events

Detect short-timescale calcium transients (not exact spikes), for example using neuron-specific thresholding such as high-SD cutoff from baseline.

  • Plot traces with event markers.
  • Create neuron-by-time scatter raster of events.
  • Extract quantitative transient features and summary metrics.

Reference on spike deconvolution limits and methods: Theis et al..

Task 1.3: Network and ensemble features

  • Compute coactivity in short time bins (e.g., 500 ms).
  • Build null distributions using circular shuffle controls.
  • Identify significant ensemble events and contributor neurons.
  • Compare events via Jaccard similarity and cluster recurring motifs.

Ensemble analysis reference: Wenzel and Hamm.
